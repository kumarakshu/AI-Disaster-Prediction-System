# AI Disaster Prediction System

## Overview
AI-powered flood prediction system using Machine Learning, FastAPI, and real-time analytics.

## Features
- Flood risk prediction using XGBoost
- FastAPI backend
- Frontend dashboard
- Database logging
- Feature importance analysis

## Tech Stack
- Python
- Scikit-learn
- XGBoost
- FastAPI
- SQLite
- HTML, JS

## How to run

### Backend
cd backend
uvicorn main:app --reload

### Frontend
open frontend/index.html
